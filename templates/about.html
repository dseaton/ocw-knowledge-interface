Overview

OpenCourseWare contains content from thousands of courses across all departments at MIT. Organizing information and providing users entrypoints is a consuming task that often requires manual tagging. Word embeddings -- a term from a class of natural language processing (NLP) techiques that encode semantic meaning in vector representations -- can provide insight into content, while providing several convenience features that can be used in experimental interfaces for users.

Word embeddings

You can learn more about word embeddings here. Our model is built with the popular NLP package gensim using their doc2vec functionality.
